# Iran Conflict Monitor

Real-time intelligence dashboard for monitoring the Iran-US/Israel military conflict.

## Features

- **Live News Feed** — 60+ verified news items from Jan 2025–Feb 2026 with source attribution, date, and links to originals
- **Interactive Strike Map** — 71 military strikes plotted on a Leaflet.js map, color-coded by attacker (US, Israel, Iran, Houthis, Hezbollah)
- **Military Asset Tracking** — 53 military asset deployments (carrier groups, air bases, missile batteries) with real-time positions
- **Multi-Source Coverage** — Government (Pentagon, CENTCOM, IAEA), news agencies (Reuters, AP, BBC, Al Jazeera), think tanks (FDD, Soufan Center), and social media
- **Source Reliability Flags** — Unconfirmed reports flagged with "UNCONFIRMED" badge
- **Reader's Notes** — Disputed or contested claims flagged with "DISPUTED" badge and expandable counterpoint sources
- **Dark/Light Mode** — Full theme support
- **Category & Source Filters** — Filter by event type (military ops, diplomacy, sanctions, nuclear sites, etc.) and source type

## Tech Stack

- Vanilla HTML/CSS/JS (no frameworks)
- Leaflet.js for interactive mapping
- Google Fonts (Inter, JetBrains Mono)
- Responsive dark/light theme with CSS custom properties

## Data

All data is real and sourced from verified reporting. Categories include:
- Military Operations (strikes, offensives, deployments)
- Diplomacy (ceasefire negotiations, nuclear talks)
- Nuclear Sites (IAEA reports, enrichment milestones, strikes on nuclear facilities)
- Sanctions (UN snapback, US maximum pressure)
- Regional Proxy Activity (Houthis, Hezbollah)
- Civilian Impact (humanitarian reports, war crimes documentation)

## Usage

Open `index.html` in any modern browser. No build step required.
